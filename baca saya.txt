Standard Penamaan : 
- standard penamaan file PHP, CSS, JS (snake_case)
- standard penamaan file image, video, dll (CamelCase)
- standard penamaan function (PHP) (snake_case)

cara pengerjaan :
jangan commit dan push tanpa ada konfirmasi karena bisa bertabrakan.

cara commit / memasukkan / mengambil data dari github di sourcetree :
memasukkan :
- stage all
- namain (folder bagian apa), (V1.x.x + 1)
- commit
- push
contoh :
(V1.0.0) -> (V1.0.1) jika di anggapnya sebagai minor update
(V1.0.0) -> (V1.1.0) jika di anggapnya sebagai major update

mengambil :
- Fetch
- Pull

yang perlu di install :
- Node JS (20.9.0 LTS) (Recommended user)
- Composer V2.6.5
- XAMPP 8.2.4 (PHP)
- Git Bash 2.42.1
- Laravel 10.x


Git bash jika tidak bisa di download bisa menggunakan
winget install --id Git.Git -e --source winget
